Q: How will you find the link in the NTNU logo top left at this page (https://www.ntnu.no)? (NB. The link, not the actual picture).

A: For just the link you would write

.js document.getElementById("home_href").href;

This means that you get the element with the id "home_href" and then with the atribute ".href". Then you have just the link and not the picture.
